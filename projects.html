My Projects

Pig Game

  • HTML
  • CSS
  • Javascript

This is an app used to play the Pig game. Using Javascript each player rolls a dice to increase their score. The player can either continue to roll the dice or hold to keep their points. The game is to 100 points by default but that can be changed by entering a new score at any time.

My favorite feature of the game is that it comes with two themes. A light theme and a dark theme. The theme of the game can be changed at any point.

Project One

  • HTML
  • Bootstrap
  • Javascript
  • Yelp and Google Maps API

This was a group project with students from my coding Bootcamp. We decided to create somewhat of a Yelp Clone. Using the Yelp API the user can search for any topic in any city and state and get ten results back. Using the Google Maps API we then display the entered city and state and place a marker for each Yelp result.

My part in the project was to implement Google Maps. To accomplish this I need to use the Geocoding and Maps JavaScript API.

GifTastic

  • HTML
  • CSS
  • Javascript
  • Giphy API

This is an app used to get 10 gifs from the Giphy API. The theme for the project are animals when the page is loaded. However, using the 'Add new search option' you can create a button for any topic.

To use the app you just need to click on any of the preset buttons for creating your own to click. Once a button has been clicked you will receive 10 still gifs with a G-rating. If you click on the gifs it will toggle a playing animation.

Crystals Collector

  • HTML
  • CSS
  • jQuery

When the game loads there will be a number to reach and four gems. Each gem has a random value assigned to it. Using these gems you are to reach the random number given and to win you need to match it exactly. It is possible to get a set of values that will not equal the random number that is given.

When you have finished the round the game will give you a new number and the gems will get new values. It will also track your wins and losses.
